## Portfolio Site built using Vite

`A simple portfolio site built using Vite, SASS and Typescript. Built to test my SASS and Typescript skills.`

## Technologies and Dependencies

```
- Vite
- SASS
- Typescript
- React
- React Router
- FontsAwesome React
```

## Installing

```
git clone <repo>
npm install 
cd dist/portfolio-site
npm run dev
```
